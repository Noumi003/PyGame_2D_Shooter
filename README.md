# Shooter Game (Pygame)

A 2D side-scrolling shooter game developed in Python using Pygame. The game includes player combat, enemy AI, multiple levels, and a tile-based world system loaded from CSV files.

# Description
This project demonstrates core game development concepts such as movement physics, collision detection, animation handling, and basic artificial intelligence. The player navigates through levels, eliminates enemies, collects items, and reaches the exit to progress.

# Features

- Player movement with gravity and jumping
- Shooting system with limited ammo
- Grenade mechanics with explosion damage
- Enemy AI with patrol and vision-based attack behavior
- Tile-based level design using CSV files
- Health system with visual health bar
- Collectible items (ammo, health, grenades)
- Parallax scrolling background
- Sound effects and background music
- Level progression system
- Start menu and restart system

# Controls

Left Arrow / Right Arrow: Move
Up Arrow or Space: Jump
S: Shoot
X: Throw grenade
ESC: Exit

# Game Mechanics

- Physics-based movement using gravity
- Collision detection with tiles and objects
- AI enemies detect player using a vision range
- Screen scrolls dynamically based on player positionPlayer dies from falling, water, or enemy damage
- Level completes when player reaches exit
